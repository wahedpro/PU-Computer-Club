Level of the Courses

Level 1: Beginner

For newcomers to competitive programming, this level covers basic programming concepts, data types, loops, and simple problem-solving skills.

  • Introduction to Data Types
  • Loops (for, while)
  • Basic Input/Output
  • Simple Array Manipulations
  • Conditional Statements (if, else)
Level 2: Intermediate

Designed for programmers with basic knowledge, this level focuses on data structures, algorithms, and medium-level problem-solving techniques.

  • Sorting and Searching Algorithms
  • Introduction to Recursion
  • Basic Data Structures (Stacks, Queues)
  • Math-based Problems
  • Two-pointer Technique
Level 3: Advanced

For experienced programmers, this level includes complex algorithms, advanced data structures, dynamic programming, and optimization techniques.

  • Dynamic Programming
  • Graph Algorithms (DFS, BFS)
  • Advanced Data Structures (Segment Trees, Fenwick Trees)
  • Greedy Algorithms
  • Bit Manipulation
