Technologies & Tools

These are the tools and technologies I use to bring games to life. From powerful game engines to design software, I ensure the highest quality in every project.

Unity

A powerful game engine used for creating both 2D and 3D games. Its flexibility and cross-platform support make it a go-to for indie and AAA game developers.

Unreal Engine

A robust game engine known for creating high-quality 3D environments and AAA games. Unreal Engine provides advanced features for both design and development.

Blender

A versatile 3D creation suite used for modeling, rigging, animation, rendering, compositing, and video editing.

Photoshop

Industry-leading software for image editing and texture creation. Photoshop is essential for designing assets such as game UI, textures, and concept art.

C#

The primary programming language used in Unity game development. C# is known for its simplicity and powerful capabilities in creating interactive gameplay experiences.

Python

A versatile and easy-to-learn programming language used for scripting, AI, and automating game development tasks like procedural generation and backend logic.
